# 劇団向け 公演掲載マニュアル（はじめての方向け）

この手順書は、**このポータルサイトに自分の劇団の公演を掲載する方法**を、できるだけやさしくまとめたものです。  
パソコン操作が苦手な方でも、画面の表示どおりに進めれば登録できます。

## 1. まず準備するもの

1. ログインに使うメールアドレス（Googleアカウントでも可）
2. 公演チラシ画像（`jpg` / `png` / `webp`、**5MB以下**）
3. 表示用画像（トップや詳細で見せる画像、`jpg` / `png` / `webp`、5MB以下）
4. 公演情報メモ
   - 公演タイトル
   - URL用英字名（例: `nights-coffee`）
   - 公演開始日時（1件以上）
   - カテゴリ（コメディ、会話劇など）

## 2. ログインする

1. ブラウザで ` /register ` を開く  
   - 例: `https://あなたのドメイン/register`
2. 「**Googleでログイン**」または「**メールリンクでログイン**」を押す
3. ログイン後、「現在の状態」が**ログイン中**になっていることを確認する

## 3. 劇団情報を登録する（初回のみ）

1. ` /register ` の「**劇団情報の登録 / 追加**」まで下に進む
2. 次の項目を入力する
   - 劇団名
   - 連絡先メール
   - 公式サイトURL（任意）
   - 劇団紹介（任意）
3. 「**劇団情報を送信**」を押す
4. 「劇団を追加しました...」の表示が出たら完了

補足:
- 同じ劇団名がすでに存在すると登録できません（名前を少し変えて再送信）。

## 4. 公演を新規登録する

1. ` /theater ` を開く
2. 「**新規公演を作成**」を押す
3. 画面上部の4ステップを順番に進める

### ステップ1: 画像

1. 「**チラシ画像**」を選んでアップロード
2. 必要なら「公演ページURL（AI解析用・任意）」を入力
3. 「**AI解析を実行**」を押す（自動入力される）
4. 「**表示画像（トップ・詳細ページ用）**」もアップロード
5. 「次へ」を押す

### ステップ2: 基本

1. カテゴリを1つ以上選ぶ（複数可）
2. 「URL用英字名（スラッグ）」を入力（例: `nights-coffee`）
3. 「公演タイトル」を入力
4. 「次へ」を押す

注意:
- 一番先頭のカテゴリが公開URLに使われます。  
  例: `/events/comedy/nights-coffee`

### ステップ3: 日程・会場

1. 「情報公開日時」（任意）を入力  
   - 未来日時を入れると、その時刻まで公開されません
2. 「公演開始日時（複数）」を**1件以上**入力
3. 必要に応じて会場名、住所、料金、販売所URLを入力
4. 「次へ」を押す

### ステップ4: 詳細・公開

1. あらすじ、タグ、脚本、演出、キャストを入力（任意）
2. 「ステータス」を選ぶ
   - `公開`: 公開ページに表示
   - `下書き`: 一般公開しない
   - `非公開`: いったん公開した公演を引っ込めるときに使用
3. 「**公演を作成**」を押す
4. 「公演を作成しました。」が出たら保存完了

## 5. 公開を確認する

1. ` /theater ` に戻る
2. 作成した公演の行で「**公開ページ**」を押す
3. 公開ページで以下を確認する
   - 日時
   - 会場
   - 料金
   - 販売ページURL
   - 画像表示

## 6. いったん下書き・非公開にする方法

1. ` /theater ` で対象公演の「**編集**」を押す
2. 編集画面上部で状態を変更する
   - 「公開する」
   - 「非公開にする」
   - 「下書きに戻す」
3. 必要ならフォーム内の内容を直し、「変更を保存」

## 7. 削除する方法（元に戻せません）

1. 公演の編集画面を開く
2. 下部の「**完全削除**」を押す
3. 確認ダイアログでOKすると削除完了

## 8. SNS宣伝文を作る方法（任意）

1. 公演を保存した後、編集フォーム下部の「SNS宣伝文生成」を開く
2. 対象SNS（twitter / instagram / facebook）にチェック
3. 「**宣伝文を生成する**」を押す
4. 「コピー」を押してSNS投稿に貼り付ける

## 9. よくあるエラーと対処

1. 「URL用英字名と公演タイトルは必須です。」
   - スラッグとタイトルが空です。両方入力してください。
2. 「公演開始日時を1つ以上入力してください。」
   - 日時が未入力です。最低1件追加してください。
3. 「カテゴリを1つ以上選択してください。」
   - カテゴリ未選択です。1つ以上チェックしてください。
4. 「同じ名前の劇団が既に登録されています」
   - 既存名と重複しています。劇団名を調整してください。
5. `file size must be 5MB or less`
   - 画像サイズが大きすぎます。5MB以下にして再アップロードしてください。
6. `Only jpg, png, webp files are allowed`
   - 画像形式を `jpg/png/webp` に変換してください。

## 10. 迷ったときの最短ルート

1. ` /register ` でログイン
2. 劇団情報を送信
3. ` /theater ` で「新規公演を作成」
4. 必須4点だけ入力  
   - カテゴリ
   - スラッグ
   - タイトル
   - 公演開始日時
5. ステータスを「公開」にして保存

以上で掲載できます。まずは1件登録して、あとから編集で整える運用がおすすめです。
